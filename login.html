Da kreiramo form, samo u html. Input tag je self-closing. U input type, ako je text, primiće text, ako je email, može primiti samo email, password će primiti lozinku i pokazat će se u tačkicama. Ako upišemo required, podaci se moraju popuniti prije nego što se formular može podnijeti. (Submit). Ako želimo da naglasimo neku riječ u formularu, možemo koristiti span tag, koji kasnije unutar css stila možemo označiti da bude boldirana, ili italic ili podvučena, druge boje, itd.
